# Chapter12. 탐색 2


기존 이진탐색트리의 문제점을 개선한 AVL 트리를 제시한다.  
AVL 트리의 핵심은 왼쪽 서브트리와 오른쪽 서브트리의 Height 의 차이가 2 이상으로 벌어지면  
LL, LR, RL, RR 리밸런싱을 통하여 각 서브트리의 밸런스를 다시 잡는 것이 핵심이다.  


이외에도 Red Black Tree, B Tree 등 이진탐색트리의 문제점을 보완한 다양한 트리가 존재하는데,  
이에 대해서는 언급만 하고 다루지는 않는다.  


근데 어찌됐든 결국 다양한 변형된 트리는 Height 차이가 크게 벌어지지 않도록 하는 것이 핵심이다.
