# Chapter14. 그래프

그래프를 Linked List 로 구현해보고, DFS 와 BFS 를 구현하고,  
최소신장트리(MST) 와 Kruskal 알고리즘을 구현하는 챕터였다.  


다른 챕터에 비해 다양한 주제를 다루는 듯한 느낌이 들었고,  
특히 최소신장트리의 경우 이전에 다뤘던 다양한 자료구조를 모두 사용하여  
구현한다는 점이 재미있었다. 그래서 그런지 마무리하는 느낌이 잘 들었던 듯 하다.  


DFS, BFS 는 그 개념은 알고있었지만 깊게 생각해볼 일이 없어서  
깊게 생각해본 적은 없었는데,  
DFS 와 BFS 를 스택과 큐로 구현해보니 그 구현에 있어 나름 흥미로운 구석이 있었다.
