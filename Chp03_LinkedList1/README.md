# Chapter3. 연결 리스트 1

추상 자료형 또는 ADT(Abstract Data Type) 이란 구체적인 기능의 완성 과정을 언급하지 않고, 순수하게 기능이 무엇인지를 나열한 것이다.  

<br>

리스트 자료구조의 ADT 는 아래처럼 정의할 수 있다.  
- void ListInit(List): 주어진 리스트를 조기화한다.
- void LInsert(List, Data): 주어진 리스트에 값을 추가한다.
- int LFirst(List, Data): List 의 첫 번째 값을 Data 에 저장한다. 성공 시 1, 실패 시 0 을 반환한다.
- int LNext(List, Data): 주어진 Data 의 다음 데이터가 Data 에 저장한다. 성공 시 1, 실패 시 0 을 반환한다.
- Data Remove(List): List 의 첫 번째 값을 List 에서 제거하고 해당 데이터를 반환한다.
- int Count(List): 리스트에 저장되어 있는 데이터 수를 반환한다.

---

메모리의 해제에 대해 리스트가 책임져야 하는가?  
- 일반적으로 리스트는 메모리의 해제까지 책임지지 않는다.
- 예를 들어 리스트에 저장된 값이 주소 값인 경우, 그리고 그 주소 값이 동적 할당의 결과인 경우를 구분하여 메모리의 해제를 진행하도록 하는 것은 무리가 있다.  
- 따라서 LRemove 같이 데이터를 없애는 함수는 소별된 데이터를 반환하도록 하여 데이터 삭제 여부를 외부에서 결정할 수 있도록 해야 한다.
