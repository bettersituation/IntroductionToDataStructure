# Chapter1. 자료구조와 알고리즘의 이해

알고리즘은 자료구조에 의존적이다.  
예를 들어 특정 배열의 원소의 합은 반복문을 사용하여 구할 수 있으나 그래프의 원소의 합은 단순한 반복문으로 구할 수 없다.  
따라서 자료구조와 알고리즘은 동전의 앞뒷면과 같다.  

<br>

잘 동작하는 자료구조와 알고리즘을 찾기 위하여 시간 복잡도와 공간 복잡도를 고려한다.  
보통 최악의 경우를 가정하여 이론적 복잡도를 계산한다.  

<br>

시간 복잡도는 연산 횟수에 의존적이고 연산 횟수는 데이터 개수 n에 의존적이다.  
따라서 시간 복잡도는 데이터 개수에 대한 함수, 특히 초등함수로 표현될 수 있으며  
그 중에서도 데이터 개수 n 이 커짐에 따라 다른 항을 무시하는 효과를 가진 항이 존재한다.  
Big-O 표기법은 계수를 제외한 해당 항을 표기하는 표기법이라고 할 수 있다.
